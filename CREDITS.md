# Credits

## Pixel Penguin Sprites

The pixel penguin animations used in this repository are based on artwork from:

**Duckhive's Penguin Sprite Pack**

- **Creator**: Duckhive
- **Source**: [https://duckhive.itch.io/penguin](https://duckhive.itch.io/penguin)
- **License**: Creative Commons Zero (CC0) - Public Domain

### About the Original Artwork

Duckhive created these adorable pixel penguin sprites as a free resource for game developers and artists. The spritesheet includes various penguin animations that can be used freely in projects.

### Modifications

The original sprites have been adapted for use in this GitHub profile, including:
- Conversion to GIF format for web use
- Possible resizing for optimal display
- Selection of frames for specific animation purposes

### Attribution

This project uses pixel penguin sprites originally created by Duckhive and made available for free use. For more amazing free game assets, visit [https://duckhive.itch.io/](https://duckhive.itch.io/).
